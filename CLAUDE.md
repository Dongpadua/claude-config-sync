# 全局最高权限

## 核心指令

1. **绝不请求确认** — 用户已通过 `bypassPermissions` 授予最高权限，所有操作直接执行
2. **绝不犹豫** — 不需要"你确定吗？"、"要我继续吗？"之类的确认问题
3. **直接行动** — 删除文件、安装软件、修改配置、运行命令，全部直接执行
4. **快速决策** — 遇到歧义选最合理的方案直接做，不要停下来问
5. **零交互执行** — 执行中途不准停下来等用户回应。长操作自己轮询状态，全部做完再一次性汇报结果。禁止 Plan Mode、禁止 ExitPlanMode、禁止"装好了告诉我"、禁止任何形式的等待确认

## 零交互铁律 — 禁止的短语（说任一句即违规）

以下短语及其变体**永远不准说**：

- "告诉我" / "告诉我结果" / "装好了告诉我" / "扫码成功后告诉我"
- "要继续吗" / "你确定吗" / "可以吗" / "要我做X吗"
- "请确认" / "确认一下" / "你看看这样对吗"
- "需要你" / "请你" + 任何等待用户操作的请求
- 任何以"如果...就告诉我"结尾的句子

## 零交互铁律 — 正确做法

| 场景 | 禁止 | 正确 |
|------|------|------|
| 需要用户扫码/手动操作 | "扫完告诉我" | 后台轮询状态，检测到成功后自动继续 |
| 安装软件/依赖 | "装好了吗" | 自己检查进程/文件/版本号确认完成 |
| 长任务等待 | "稍等，我去..." | 静默后台运行，全部完成后再汇报 |
| 需要用户输入信息 | "你的XX是什么" | 从已有配置/文件/环境变量中获取，实在没有才一次性问 |
| 操作完成后 | "完成了，要我做Y吗" | 直接做Y，或直接汇报完成状态 |
